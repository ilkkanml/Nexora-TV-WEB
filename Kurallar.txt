

Önce kuralı sabitleyelim: **Nexora TV paneli içerik/provider/playlist satmaz ve saklamaz.** Kullanıcı kendi yetkili kaynak bilgisini uygulama içinde/local kullanır; backend sadece platform, cihaz, erişim, lisans, sürüm, destek ve operasyon verisini yönetir. Bu sınır net: ürün channel, playlist, hosted stream, third-party content veya provider access sağlamaz.

```txt

```

İlişki mantığı:

```txt
users.id
roles.id
permissions.id
role_permissions.role_id → roles.id
role_permissions.permission_id → permissions.id
user_roles.user_id → users.id
user_roles.role_id → roles.id

resellers.id
reseller_staff.reseller_id → resellers.id
reseller_staff.user_id → users.id

customers.user_id → users.id
customers.reseller_id → resellers.id nullable

devices.user_id → users.id nullable
devices.reseller_id → resellers.id nullable
devices.install_id → device_install_records.installId

licenses.user_id → users.id
licenses.reseller_id → resellers.id nullable
licenses.device_id → devices.id nullable
licenses.plan_id → plans.id

activation_keys.created_by_user_id → users.id
activation_keys.assigned_user_id → users.id nullable
activation_keys.reseller_id → resellers.id nullable
activation_keys.license_id → licenses.id nullable

support_tickets.user_id → users.id
support_tickets.reseller_id → resellers.id nullable
support_messages.ticket_id → support_tickets.id

audit_logs.actor_user_id → users.id
```

